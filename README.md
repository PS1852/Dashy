# Dashy | Your Digital Workspace

![Dashy Logo](public/logo.png)

A premium, Notion-inspired productivity dashboard built with React, TypeScript, and Appwrite. Dashy provides a clean, minimal interface for organizing your thoughts, tasks, and projects.

## ✨ Features

- **Notion-Inspired UI**: Clean, ivory-and-ink aesthetic with "The Digital Atelier" design system.
- **Google Authentication**: Seamless login with Google OAuth 2.0.
- **Real-time Sync**: Powered by Appwrite Cloud for instant data persistence.
- **TypeScript Core**: Fully typed codebase for robust performance.
- **Modern Stack**: Built with Vite + React + Lucide Icons.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/PS1852/Dashy.git
   cd Dashy
   ```

2. **Install dependencies**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

## 🛠 Tech Stack

- **Frontend**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Backend**: [Appwrite Cloud](https://appwrite.io/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS (Modern CSS Variables)

## 🎨 Design System: The Digital Atelier

Dashy follows a specific design philosophy:
- **Ivory Background**: Soft #FBF9F7 for premium eye-feel.
- **Tonal Layering**: Depth via color shifts instead of rigid borders.
- **Inter Typography**: professional, geometric sans-serif.
- **Minimalist Iconography**: Purposeful use of Lucide icons.
